# Collectif Jardin — Instructions permanentes

## Projet
Site web vitrine d'un collectif de musique électronique à Rimouski.
Fichier principal : index.html
Déploiement : GitHub Pages sur collectifjardin.org

## Stack
HTML5 + CSS3 + JS vanilla — fichiers statiques uniquement
Aucun framework, aucune dépendance externe, aucun build tool

## Identité visuelle — NON NÉGOCIABLE
- --bg: #141414
- --mauve: #edacff
- --orange: #fd2f00
- --burgundy: #38050e
- --cream: #fffbdb
- Polices : Cormorant Garamond (titres) + Space Mono (texte)
- Style : sombre, underground, élégant, minimaliste

## Priorités
1. Mobile-first — penser 390px en premier, puis 1440px
2. iOS Safari 15+ est la cible prioritaire
3. Ne jamais supprimer de contenu sans confirmation explicite
4. Ne jamais modifier la charte graphique
5. Ne jamais introduire de frameworks ou dépendances
6. Les animations existantes sont gelées sauf demande explicite
